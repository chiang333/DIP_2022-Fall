# HW2
## 1. Contrast Enhancement
### COMPILE
```
g++ -std=c++14 contrast.cpp -o contrast
```

### EXECUTE
```
./contrast
```

### INPUT FILE
input file should be in the same folder named `input1.bmp`

### OUTPUT FILE
`output1_1.bmp` (gamma enhancement)  
`output1_2.bmp` (histogram enhancement)
